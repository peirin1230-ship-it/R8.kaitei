## 2. 施設基準の変更と届出対応

### 2-1. 地域支援・医薬品供給対応体制加算（旧：後発医薬品使用体制加算）

**届出の切替え対応が必要**

| 要件 | R6 | R8 |
|------|-----|-----|
| 後発品使用割合 | 加算1：90%以上、2：85%以上、3：75%以上 | **据置** |
| 先発+後発≧50%の下限要件 | あり | **廃止** |
| ジェネリック推進掲示 | なし | **院内の見やすい場所に掲示必須** |
| 供給不足時の処方変更対応体制 | なし | **体制整備必須** |
| 薬価差を考慮しない価格交渉の実施 | なし | **単品単価交渉の実施** |
| ホームページ公表 | なし | **掲示内容のWeb公開必須** |
| 卸への過度な要求の抑制 | なし | **頻回配送・休日夜間配送・緊急配送の過度な要求禁止** |
| 温度管理品等の返品抑制 | なし | **温度管理品・在庫調整返品の抑制** |
| 地域の医療機関・薬局との情報共有 | なし | **供給情報の共有に協力** |

### 2-2. バイオ後続品使用体制加算

**対象バイオ医薬品リストの大幅拡大**

| グループ | 閾値 | R6 | R8 |

## 4. 算定上の留意事項

### 4-1. 薬剤適正使用連携加算の拡大

| | R6 | R8 |
|---|---|---|
| 対象 | 入院・入所患者の転院・退所時のみ | **外来での継続管理にも拡大** |
| 算定期間 | 退院後2か月以内 | **退院後3か月以内** |
| 情報共有 | 「連携」 | **「情報提供」（双方向）** |

### 4-2. 在宅自己注射指導管理料の対象薬剤追加

| 新規追加薬剤 | 条件 |
|-------------|------|
| ベドリズマブ製剤 | 皮下注のみ |
| ミリキズマブ製剤 | 皮下注のみ |
| ベンラリズマブ製剤 | EGPA患者のみ |
| クロバリマブ製剤 | 皮下注のみ |
| ロペグインターフェロンアルファ製剤 | 真性多血症、皮下注 |

### 4-3. 在宅麻薬等注射指導管理料（C108）

- **腎不全**が対象疾患に追加（心不全・呼吸器疾患と並列）
- フェンタニルクエン酸塩製剤が新たに対象薬剤に追加

### 4-4. 経腸投薬用ポンプ加算（C152-3）

- R6：2か月に2回まで → R8：**回数制限なし**

### 4-5. 生活習慣病管理料とC101の併算定ルール変更

- R6：C101算定時は生活習慣病管理料の算定不可（一律）
- R8：糖尿病が主病で**別に定める薬剤**を投与中の場合のみ併算定不可（対象が限定）
- 生活習慣病管理料(II)から特定薬剤治療管理料が包括除外（別途算定可に）

### 4-6. 院内投薬・院外処方の同日併用

- R8通知で「緊急やむを得ない事態」の定義が初めて詳細に規定
