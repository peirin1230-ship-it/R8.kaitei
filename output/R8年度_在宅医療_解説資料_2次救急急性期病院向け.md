> **出典**: 診療報酬の算定方法の一部を改正する件（令和8年厚生労働省告示第69号）医科点数表、同実施上の留意事項について（令和8年3月5日保医発0305第6号）
>
> **対象読者**: 2次救急指定の急性期病院における医事課・経営企画部門

> **本資料の範囲**: 医科点数表 第2章第2部「在宅医療」のうち、急性期病院が算定する項目に限定。往診料・訪問診療料・在宅時医学総合管理料等のかかりつけ医が算定する項目は除外している。

## 第2章 救急搬送関連

### 2.1 C004 救急搬送診療料

| 項目 | 点数 |
|---|---|
| **本体** | **1,300点** |
| 注2 新生児加算 | 1,500点 |
| 注2 乳幼児加算（6歳未満・新生児除く） | 700点 |
| 注3 長時間加算（診療開始〜搬送先到着が30分超） | 700点 |
| 注4 重症患者搬送加算 | 1,800点 |

#### 算定要件

- 患者を救急用自動車（当該保険医療機関に属するもの）又は救急医療用ヘリコプターで搬送する際に、**医師が同乗して診療を行った場合**に算定する。
- 患者の発生現場に赴き診療を行い、その後救急車に同乗して搬送した場合は、**往診料と併算定可能**。

#### 算定できない場合

- 入院患者を他医療機関に搬送した場合は**原則算定不可**。ただし以下は例外：
  - 搬送元以外の医療機関の医師が同乗した場合
  - 救急搬送中に**人工心肺補助装置、補助循環装置又は人工呼吸器**を装着し、医師による集中治療を要する状態の患者について、日本集中治療医学会の定める指針等に基づき搬送を行う場合

#### 注4 重症患者搬送加算の要件

- **人工心肺補助装置、補助循環装置又は人工呼吸器**を装着し、医師による集中治療を要する状態の患者
- **重症患者搬送チーム**（医師、看護師、臨床工学技士等）が搬送した場合に算定

#### R6→R8の変更点

- 直接的な点数変更なし（R6から据え置き）

### 2.2 C004-2 救急患者連携搬送料 【R8大幅改正】

R8改定で最も大きく変わった項目の一つ。搬送先の算定（「2」）が新設され、搬送方法の柔軟化と長時間加算が追加された。

#### 点数体系

**1 救急患者連携搬送料1（搬送元が算定）**

| 区分 | 入院外 | 入院初日 | 入院2日目 | 入院3日目 |
|---|---|---|---|---|
| **イ 医師・看護師又は救急救命士が同乗** | 2,400点 | 1,200点 | 800点 | 600点 |
| **ロ その他** | 1,000点 | 500点 | 350点 | 200点 |

**2 救急患者連携搬送料2（搬送先が算定）【新設】**

| 区分 | 点数 |
|---|---|
| **イ 医師等が同乗して搬送する場合** | 800点 |
| **ロ その他の場合** | 200点 |

**加算**

| 項目 | 点数 | 要件 |
|---|---|---|
| 注5 長時間加算【新設】 | 700点 | 搬送元出発〜搬送先到着が30分超（1イ又は2イの場合のみ） |

#### 算定要件（搬送元：「1」）

- 救急外来を受診した患者に**初期診療を実施**した後、連携する他の医療機関での入院が適当と判断して搬送した場合に算定する。
- 搬送先：**特定機能病院、救命救急センター、高度救命救急センター**は除外。「A200急性期総合体制加算」届出医療機関への搬送も除外。**特別の関係にある保険医療機関**への搬送も算定不可。
- 搬送後に搬送先に**入院しなかった場合は算定不可**。

#### 算定要件（搬送先：「2」）【新設】

- 連携先医療機関から搬送を受け入れ、**入院させた場合に入院初日に限り**算定する。

#### 搬送方法

- **イ（医師等同乗）**: 緊急自動車（当該医療機関又は搬送先に属するもの）で搬送。患者からの搬送費用徴収不可。
- **ロ（その他）**: 救急隊の救急自動車以外での搬送も可能。認定業者の搬送車等も使用可。

#### R6→R8の主要変更点

| 変更内容 | R6 | R8 |
|---|---|---|
| 搬送先の算定 | なし | **「2」新設**（イ800点/ロ200点） |
| 搬送元区分 | 一律（1,800/1,200/800/600点） | **医師同乗/その他の2区分に細分化** |
| 長時間加算 | なし | **注5新設**（700点） |
| 搬送方法 | 緊急自動車のみ | **救急隊以外の搬送も可能** |
| 搬送先除外対象 | A200+A200-2 | A200急性期総合体制加算 |

#### 2次救急病院での活用場面

1. **搬送元として（「1」を算定）**: 自院の救急外来で初期診療後、専門的治療が必要な患者を連携先の急性期病院に転院搬送する場合。
2. **搬送先として（「2」を算定）**: 連携先の1次救急や他の2次救急から転院搬送された患者を入院させた場合。入院初日に限り算定。

## 第3章 退院支援・訪問看護指示関連

### 3.1 C005-2 在宅患者訪問点滴注射管理指導料

| 項目 | 点数 |
|---|---|
| **本体**（1週につき） | **100点** |

#### 算定要件

- 在宅で療養を行っている通院困難な患者に対し、**週3日以上の点滴注射**を行う必要を認め、当該患者の訪問看護を担う訪問看護ステーション等に点滴注射に係る指示を行った場合に算定。
- 算定は**週1回**、月は関係なく連続した7日間で判断する。

#### R6→R8の変更点

- 衛生材料・保険医療材料の供与方法に「第2節第1款の2又は3による方法で供与することでもよい」を追加。
- **看護師等遠隔診療注射実施料との併算定不可**を新設。

### 3.2 C007 訪問看護指示料

| 項目 | 点数 |
|---|---|
| **本体** | **300点** |
| 注2 特別訪問看護指示加算 | 100点（月1回。気管カニューレ使用者・重度褥瘡患者は月2回） |
| 注3 手順書加算 | 150点（6月に1回） |
| 注4 衛生材料等提供加算 | 80点（月1回） |

#### 算定要件

- 在宅療養中の**通院困難な患者**に対し、主治医が**訪問看護指示書**（有効期間6月以内）を交付した場合に算定する。
- **退院時に1回算定可**。在宅療養中は月1回限度。
- 同一月に複数の訪問看護ステーションに指示書を交付しても**月1回のみ**。

#### 特別訪問看護指示加算（注2）の要件

- **急性増悪、終末期、退院直後**等で週4回以上の頻回訪問看護が一時的に必要な場合に交付する。
- 指示日から**14日以内**に実施。
- 急性期病院では**退院直後の患者**に対して活用する場面が多い。

#### 手順書加算（注3）の要件

- **特定行為**（気管カニューレ交換、胃ろうカテーテル交換、褥瘡壊死組織除去等7項目）に係る手順書を交付した場合。
- 患者1人につき**6月に1回**限度。

#### R6→R8の変更点

- 直接的な点数変更なし。

## 第4章 在宅療養指導管理料（C100〜C121）

### 通則（共通ルール）

在宅療養指導管理料を算定する際の共通ルールは以下の通り。

| ルール | 内容 |
|---|---|
| 算定回数 | **月1回限度**。同一患者に同月2回以上行った場合は第1回で算定 |
| 複数管理料 | 同一医療機関で2以上の指導管理を行う場合は**主たる管理料の所定点数のみ算定** |
| 複数医療機関 | 2以上の医療機関が同一患者に算定する場合、**主たる医療機関で算定** |
| 入院中の算定 | **退院日に1回限り算定可**（死亡退院・転院の場合は算定不可） |
| 衛生材料 | 消毒薬、ガーゼ、注射器等は保険医療機関が提供。費用は所定点数に含まれる |
| 外来感染対策向上加算 | 6点（月1回。施設基準を満たす場合） |
| 発熱患者等対応加算 | 20点（月1回） |
| 連携強化加算 | 3点（月1回） |
| サーベイランス強化加算 | 1点（月1回） |
| 抗菌薬適正使用体制加算 | 5点（月1回） |

> **急性期病院での注意**: 退院日に算定する場合、入院中に退院後の在宅療養に関する十分な指導を行い、使用する機器や材料の説明を行ったうえで算定する。退院できなかった場合・外泊後転院した場合は算定不可。

### C100 退院前在宅療養指導管理料

| 項目 | 点数 |
|---|---|
| **本体** | **120点** |
| 注2 乳幼児加算（6歳未満） | 200点 |

#### 算定要件

- 入院中の患者が**外泊する際**に、退院後の在宅療養指導管理を行った場合に算定する。
- **外泊の初日1回に限り**算定。
- 1泊2日（入院料上は外泊でない場合）でも算定要件を満たせば算定可。

#### 他の管理料との関係

- 同一月に他の在宅療養指導管理料を算定可能。ただし**同一日**には他の管理料・材料加算は算定不可。
- 退院できなかった場合、外泊後転院した場合は算定不可。

### C101 在宅自己注射指導管理料

| 項目 | 点数 |
|---|---|
| 1 複雑な場合 | 1,230点 |
| 2 イ 月27回以下 | 650点 |
| 2 ロ 月28回以上 | 750点 |
| 注2 導入初期加算 | 580点（初回から3月限度） |
| 注3 処方変更時の導入初期加算 | 変更日から1月限度、1回に限り |
| 注4 バイオ後続品導入初期加算 | 150点（初回処方日から3月限度） |
| 注5 情報通信機器を用いた場合 | 1:1,070点 / 2イ:566点 / 2ロ:653点 |

#### 算定要件

- **対象疾患・薬剤**: インスリン製剤（糖尿病）、インターフェロンベータ（多発性硬化症）、インターフェロンアルファ（C型肝炎）、アドレナリン製剤（アナフィラキシー既往）、成長ホルモン製剤、GLP-1受容体作動薬等。
- **導入要件**: 入院又は2回以上の外来等で十分な教育期間を経て指導が必要（アドレナリン製剤は除く）。
- **「1 複雑な場合」**: 間歇注入シリンジポンプを用いて在宅自己注射を行っている患者に指導管理を行った場合。

#### 併算定不可

- 外来腫瘍化学療法診療料・外来化学療法加算を算定する患者の自己注射に係る管理料は算定不可。

#### 急性期病院での活用

- 糖尿病でインスリン導入する患者の退院時算定が最も一般的。
- 導入初期加算（580点）を退院月から3月間算定できるため、退院直後の外来フォローでも算定可能。

### C103 在宅酸素療法指導管理料

| 項目 | 点数 |
|---|---|
| 1 チアノーゼ型先天性心疾患の場合 | 520点 |
| 2 その他の場合 | 2,400点 |
| 注2 遠隔モニタリング加算 | 150点×月数（2月限度） |

#### 対象患者

- **チアノーゼ型先天性心疾患**: ファロー四徴症、大血管転位症等で発作的にチアノーゼを起こす患者。
- **高度慢性呼吸不全**: 動脈血酸素分圧55mmHg以下、又は60mmHg以下で睡眠時・運動負荷時に著しい低酸素血症を来す者。
- **慢性心不全**: NYHA III度以上で睡眠時チェーンストークス呼吸がみられ、SAS等の診断でASVが適応でない者。
- **重度群発頭痛**: 群発期間中の患者。

#### 備品義務

- 酸素吸入設備、気管挿管器具、レスピレーター等を備えていること。

#### 遠隔モニタリング加算の要件

- COPD（病期III期以上）の患者で、前回対面から今回対面までの期間、遠隔モニタリングを実施した場合。2月限度。

## 第6章 施設基準と届出

### 6.1 救急搬送診療料の施設基準

- 特段の施設基準届出は不要（保険医療機関であれば算定可能）。
- ただし、**注4 重症患者搬送加算**は「別に厚生労働大臣が定める施設基準に適合する保険医療機関」が要件。

### 6.2 救急患者連携搬送料の施設基準

- 搬送元・搬送先とも、**事前の連携体制の構築**が必要。
- R8改定で新たに以下が追加：
  - **(カ) 在宅医療関係者と救急医療関係者の協議**: 在宅療養等に関する救急搬送についての情報共有ルールの策定。

### 6.3 在宅療養指導管理料に関連する施設基準

大部分の在宅療養指導管理料は**特段の届出不要**で算定可能だが、以下は施設基準を満たす必要がある：

| 項目 | 施設基準の概要 |
|---|---|
| C101 注2 導入初期加算 | 「複雑な場合」は間歇注入シリンジポンプ使用が要件 |
| C102 注3 / C102-2 注3 遠隔モニタリング加算 | 情報通信機器を用いたモニタリング体制 |
| C103 注2 遠隔モニタリング加算 | COPD病期III期以上の患者への遠隔モニタリング体制 |
| C107-2 注2 充実管理体制加算 | 持続陽圧呼吸療法の充実した管理体制（R8新設） |

## 第7章 実務への影響と対応

### 7.1 退院支援フローにおける算定タイミング

```
入院中
  │
  ├─ 外泊時 → C100 退院前在宅療養指導管理料（120点）
  │              ※外泊の初日1回に限り
  │
  ├─ 退院日 → 在宅療養指導管理料（C101〜C121）
  │              ※退院日に1回限り算定可
  │              ※死亡退院・転院は不可
  │              ※同一日にC100との併算定不可
  │
  └─ 退院時 → C007 訪問看護指示料（300点）
                 ※退院時に1回算定可
                 ＋ 特別訪問看護指示加算（100点）
                   ※退院直後の頻回訪問が必要な場合
```

### 7.2 救急搬送時の算定フロー

```
【自院が搬送元の場合】

  救急外来受診 → 初期診療実施 → 転院搬送の判断
       │
       ├─ 医師等が同乗して緊急自動車で搬送
       │     → C004-2「1」イ（2,400/1,200/800/600点）
       │     ＋ 30分超なら長時間加算（700点）
       │
       └─ その他の方法で搬送
             → C004-2「1」ロ（1,000/500/350/200点）

【自院が搬送先の場合】

  連携先から転院搬送を受け入れ → 入院させた場合
       │
       ├─ 医師等が同乗して搬送された場合
       │     → C004-2「2」イ（800点）※入院初日のみ
       │
       └─ その他の方法で搬送された場合
             → C004-2「2」ロ（200点）※入院初日のみ

【救急車で搬送する場合（搬送元から搬送先へ直接）】

  自院の救急車に医師が同乗して診療
       → C004 救急搬送診療料（1,300点）
       ＋ 30分超なら長時間加算（700点）
       ＋ 該当すれば重症患者搬送加算（1,800点）
```

> **C004とC004-2の違い**: C004は自院の救急自動車で搬送し医師が車内で診療した場合。C004-2は連携先医療機関との転院搬送で、搬送元・搬送先それぞれが算定。
