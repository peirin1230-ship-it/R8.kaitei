## 第1章 R8改定の全体像

### 1.1 処置の構成

医科点数表における処置は、第2章第9部に位置し、以下の4節で構成される。

| 節 | 内容 |
|---|---|
| 第1節 | 処置料（J000〜J129-4） |
| 第2節 | 処置医療機器等加算（J200〜J201） |
| 第3節 | 薬剤料（J300） |
| 第4節 | 特定保険医療材料料（J400） |

### 1.2 R8改定の主要ポイント

1. **穿刺系・呼吸器系処置の一律約20%引き上げ**（約25項目）
2. **人工腎臓は全区分一律-20点**だが、腎代替療法診療体制充実加算（+20点）の新設とセットで実質据え置き
3. **新設4項目**: 電気インピーダンス断層撮影、同種死体移植腎機械灌流保存、腎代替療法診療体制充実加算、間歇的導尿の親水性カテーテル加算
4. **看護師等遠隔診療処置実施料**の通則新設（100点/150点）
5. **「気管内挿管」→「気管挿管」** への用語統一
6. **適応疾患の追加**: 血球成分除去療法に敗血症、吸着式血液浄化法に特発性肺線維症急性増悪、高気圧酸素治療の回数上限緩和

## 第2章 通則

### 2.1 時間外・休日・深夜加算

2次救急の急性期病院では、緊急処置の時間外加算が算定収入に大きく影響する。

#### 加算1（所定点数 1,000点以上の処置）

| 加算区分 | 加算率 |
|---|---|
| 休日加算1 | 所定点数 × 160% |
| 時間外加算1（入院中の患者以外） | 所定点数 × 80% |
| 深夜加算1 | 所定点数 × 160% |

#### 加算2（所定点数 150点以上の処置。加算1に該当する場合を除く）

| 加算区分 | 加算率 |
|---|---|
| 休日加算2 | 所定点数 × 80% |
| 時間外加算2（入院中の患者以外） | 所定点数 × 40% |
| 深夜加算2 | 所定点数 × 80% |

#### 留意事項

- **入院患者の休日・深夜加算1**: 病状急変による1,000点以上の緊急処置の場合のみ算定可。
- **処置の開始時間**: 直接施療した時点で判定。日をまたいだ場合は初日のみ加算算定。
- **夜間・早朝等加算との関係**: 夜間・早朝等加算を算定する初診・再診に引き続く場合は、時間外加算の対象外。
- **所定点数の範囲**: 処置料+各注加算。第2節〜第4節の費用は含まない。

### 2.2 その他の通則

| 通則 | 内容 |
|---|---|
| 対称器官（通則6） | 「片側」「1肢につき」等の記載がある処置が対象。両眼に異なる処置を行った場合は別々に算定可 |
| 基本診療料に含まれる処置（通則3） | 浣腸、注腸、吸入、100cm2未満の第1度熱傷処置、100cm2未満の皮膚科軟膏処置、洗眼、点眼、点耳、簡単な耳垢栓除去、鼻洗浄、狭い範囲の湿布処置等は**別途算定不可** |
| 耳鼻咽喉科乳幼児処置加算（通則7） | 60点/日（J095〜J115-2、6歳未満） |

#### 算定上の注意

- **「1」の算定対象**: 入院中の患者以外、及び手術後の患者（入院中に限る。手術日から**14日限度**）のみ。
- **面積合算ルール**: 同一疾病で創傷処置・皮膚科軟膏処置・湿布処置を行った場合は**面積を合算して1つの区分で算定**。併算定不可。
- **手術後の創傷処置**: 回数にかかわらず**1日につき所定点数のみ**。複数部位は面積合算。
- **中心静脈注射等の穿刺部位のガーゼ交換**: 別に算定不可。
- **在宅管理料との関係**: C109（在宅寝たきり患者）、C112（在宅気管切開）等の算定患者は算定不可（熱傷を除く）。

### 12.2 ドレナージの算定フロー

```
ドレナージ開始
  │

  │    J019 持続的胸腔ドレナージ: 990点
  │    J020 胃持続ドレナージ: 50点
  │    J021 持続的腹腔ドレナージ: 550点
  │
  ├─ 2日目以降 → J002 ドレーン法で算定
  │    持続的吸引: 50点/日
  │    その他: 25点/日
  │
  └─ 手術と同一日 → ドレナージは別算定不可
       手術翌日以降 → J002で算定可
```
